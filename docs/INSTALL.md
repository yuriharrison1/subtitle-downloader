# Installation Guide

## Automatic Installation

Run the installer:
```bash
./scripts/install.sh
```

## Manual Installation

### Dependencies

**Ubuntu/Debian:**
```bash
sudo apt update
sudo apt install python3 python3-pip python3-pyqt5 libnotify-bin
pip3 install --user subliminal
```

**Fedora:**
```bash
sudo dnf install python3 python3-pip python3-qt5 libnotify
pip3 install --user subliminal
```
